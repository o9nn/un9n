- The 4D analog of the tetrahedron (3-simplex)

- Has **5 vertices**, **10 edges**, **10 triangular faces**, and **5 tetrahedral cells**

## 7. Unfolding the 5-Cell

### 7.1 Net of the 5-Cell

A 5-cell can be unfolded into 3D space as a **net** of 5 tetrahedra. The System 5 diagram represents one such unfolding:

```
                    [Cerebral Tetrahedron]
                           /\
                          /  \
                         /    \
                        /      \
                       /________\
                      /\        /\
                     /  \      /  \
                    /    \    /    \
                   /      \  /      \
                  /________\/________\
           [Somatic]    [Integration]    [Autonomic]
                          |
                          |
                    [Spinal Column]
```

### 7.2 Folding Instructions

To fold the 5-cell back into 4D:

1. Start with the central tetrahedron (A-B-C-E)
2. Fold the Spinal tetrahedron (D) up through the center
3. Each face of the central tetrahedron becomes shared with an adjacent cell
4. The 5 cells share faces in 4D, creating a closed 4-polytope

### 7.3 Dual Polytope

The dual of the 5-cell is another 5-cell (self-dual). This means:

- Each vertex of the original corresponds to a cell of the dual
- Each cell of the original corresponds to a vertex of the dual
- The 5 triads (Cerebral, Somatic, Autonomic, Spinal, Integration) are dual to the 5 tetrahedral cells

## 8. Implications for Rn and Pk Flows

### 8.1 Rn Flows as Edge Traversals

Relational Wholes (Rn) can be understood as **bidirectional edge traversals** in the 5-cell:

- **R1 (Forward)**: Traverse edge in one direction
- **R2 (Backward)**: Traverse edge in opposite direction
- **Countercurrent Balance**: R1 and R2 balance across each edge

### 8.2 Pk Flows as Face Projections

Projections (Pk) can be understood as **unidirectional face projections**:

- **P1 (Input)**: Project through one face into the cell
- **P2 (Output)**: Project through another face out of the cell
- **Balance Sheet**: Compare P1 and P2 across the cell

### 8.3 Cell Coherence

Each tetrahedral cell represents a **coherent subsystem**:

- The 4 vertices of a cell work together as a unit
- Rn flows within the cell create internal balance
- Pk flows between cells create external accounting
